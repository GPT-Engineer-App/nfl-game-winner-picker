# nfl-game-winner-picker

An app where users have to select the winner of every NFL game before a season starts. Once the season starts you get a point for each ocrrect selection. And a scoreboard will show in real time who the current leader is in terms of correct selections.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/nfl-game-winner-picker.git
cd nfl-game-winner-picker
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
